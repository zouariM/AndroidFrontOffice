Le projet est composée par deux entités: Back-End et Front-End
-- Installation de Back:
1-Installation d'un serveur mongo et lancement de serveur
2- Build jar file par mvn clean install (en mode sudo)
3- Lancement de serveur Http par java -jar "target/fichier.jar"

-- Installation de Front:
Lancement avec android-studio et ajout des fichiers jar de API Http-Core et Http-Client.
